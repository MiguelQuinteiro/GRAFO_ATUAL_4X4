# GRAFO_ATUAL_4X4 🕸️

Este repositorio contiene el motor de visualización y análisis de **Teoría de Grafos** aplicado a estructuras de 4x4, desarrollado en Visual Basic 6.0.

## 🚀 Descripción

`GRAFO_ATUAL_4X4` es una herramienta especializada en representar las relaciones entre los elementos de una matriz 4x4 como un sistema de nodos y aristas. El proyecto permite mapear las dependencias y flujos dentro de la grilla, facilitando el estudio de simetrías y caminos críticos en soluciones matemáticas complejas.

Es un componente ideal para visualizar cómo se "conectan" los números dentro de una plantilla (*template*) de Sudoku o cualquier sistema matricial de cuarto orden.

## 🛠️ Especificaciones Técnicas

- **Lenguaje:** Visual Basic 6.0.
- **Arquitectura:** Motor de renderizado de grafos con lógica de interconexión dinámica.
- **Funcionalidades Clave:**
  - Representación de celdas como nodos en un espacio bidimensional.
  - Generación de aristas basadas en reglas de adyacencia o lógica matemática.
  - Interfaz gráfica para la manipulación y movimiento de los componentes del grafo.
  - Algoritmos de recorrido de grafos aplicados a la estructura 4x4.

## 📂 Estructura del Repositorio

Tras la reestructuración profesional:
- `/src`: Formularios de visualización (.frm), clases de nodos (.cls) y módulos de grafos (.bas).
- `/docs`: Teoría aplicada de grafos y esquemas de conexión.
- `/assets`: Recursos visuales para la representación de los nodos.
